Address Classification for Logistics Company - LWE

We have implemented an address delivery classification system that classifies address to its property type based only on text-based patterns. This was done without address normalization and cleaning out redundancies, errors and re-duplication. The system achieves around 80% accuracy. We also propose a bootstrapped continuous training process so that the model can be more accurate the longer it is being used. The objective for such classification is to be able to predict property type for route optimization for logistic delivery. To our knowledge this is the first description of such application.

Read the published paper here https://www.researchgate.net/publication/336587926_A_Two-Stage_Text-based_Approach_to_Postal_Delivery_Address_Classification_using_Long_Short-Term_Memory_Neural_Networks

Concept Timestamping on Blockchain and Decentralization of Patents



We describe a system for registering new inventions and design based on conceptual graphs and the blockchain. Instead of describing the innovation in text as in patent claims, the claims are documented using con- ceptual graphs as a structured graphical method for capturing concepts and their relationship. Conceptual graphs as a structured graphical method allows easier automatic semantic comparisons. It obviates the need from extracting semantic concepts from text using natural language processing techniques. The graphs then can be hashed and submitted to the blockchain for timestamping to claim the innovation.

https://www.researchgate.net/publication/337035795_Concept_Timestamping_on_Blockchain_and_Decentralization_of_Patents
